GitHub Sync Setup

Your data will be saved to a file called poketracker-data.json in your GitHub repo. Pull to load data on any device. Push to save changes back.
To create a token: github.com/settings/tokens/new → select repo scope → Generate token. Your token is stored only in this browser's localStorage and is never sent anywhere except GitHub's own API.
