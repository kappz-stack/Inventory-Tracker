📱 Import from Mac Numbers

PokéTracker can't read .numbers files directly — Apple uses a proprietary format. Export your file first using one of these two methods:
Option 1 — Export as Excel (recommended)
  1. Open your file in Numbers
  2. Click File → Export To → Excel…
  3. Keep default settings, click Next
  4. Save the .xlsx file
  5. Click Import below and choose that file
Option 2 — Export as CSV
  1. Open your file in Numbers
  2. Click File → Export To → CSV…
  3. This exports one sheet at a time — export each tab separately
  4. Click Import below and choose the CSV file
💡 Tip: The Excel export keeps all 4 sheets in one file and is the easiest option. If you use Numbers regularly, keep a copy of the exported .xlsx and just re-export when you want to import changes.
